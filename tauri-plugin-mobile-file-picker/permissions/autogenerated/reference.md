## Default Permission

Default permissions for the mobile-file-picker plugin

#### This default permission set includes the following:

- `allow-pick-file`
- `allow-pick-directory`
- `allow-read-content`
- `allow-copy-to-local`
- `allow-write-content`
- `allow-release-access`
- `allow-ping`
